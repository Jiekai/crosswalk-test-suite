Precondition

  • Cordova Android 4+, CLI 5+
    $ sudo npm install cordova -g
  • minSdkVersion 16 (4.1 is the minimum supported version)
  • Set the environment path to the Android SDK Platform-tools and Android SDK Build-tools
  • Install Apache Ant
  • Set the JAVA_HOME to the environment path
Refer to https://github.com/mapsplugin/cordova-plugin-googlemaps/wiki/Tutorial-for-Mac

Test Steps

  1. Obtain the Google Maps API key for Android
    • Locate your debug keystore file: debug.keystore
      OS X and Linux: ~/.android/
    • List the SHA-1 fingerprint
      keytool -list -v -keystore ~/.android/debug.keystore -alias androiddebugkey -storepass android -keypass android
    • Go to https://console.developers.google.com
    • Select a project, or create a new one
    • Open the API Library(https://console.developers.google.com/projectselector/apis/library) in the Google Developers Console. If prompted, select a project or create a new one.
      Select the Enabled APIs link in the API section to see a list of all your enabled APIs. Make sure that the API is on the list of enabled APIs.
      If you have not enabled it, select the API from the list of APIs, then select the Enable API button for the API.
      The only API you need is the Google Maps Android API, although you can choose to enable other APIs for the same project too.
    • In the sidebar on the left, select Credentials
    • If your project doesn't already have an Android API key, create one now by selecting
      Add credentials > API key > Android key
    • In the resulting dialog, enter your app's SHA-1 fingerprint, and please fill the package name with com.example.hellomap
    • Your new Android API key appears in the list of API keys for your project. An API key is a string of characters, something like this:
      AIzaSyBdVl-cTICSwYKrZ95SuvNw7dbMuDt1KG0
  2. Build
    • cd sampleapp/res/GoogleMapsPlugin
    • ./google-maps-plugin.py -k [Google Maps API key]

    3. Details https://github.com/mapsplugin/cordova-plugin-googlemaps/wiki/Tutorial-for-CrossWalk-Webview-Plugin-%28Android%29

  3. Install and Run
    • The apk can be installed and uninstalled successfully
    • Launch the app, the google map load successfully
    • Click full screen, the google map should change to full screen, full screen button is hiddened
